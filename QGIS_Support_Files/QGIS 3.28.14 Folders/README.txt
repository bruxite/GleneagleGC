To open the heightmap in the Unity "Scene" window, double-click "COURSENAME_Lidar_Scene"
located in the Unity "Project" window.

Package File list:
  Terrain Backup (Folder)
    COURSENAME_Lidar_Inner             (Inner Terrain file)
    COURSENAME_B                       (Inner BING Terrain satellite overlay)
    COURSENAME_G                       (Inner GOOGLE Terrain satellite overlay file)
    COURSENAME_Lidar_Surface_Inner     (Inner terrain elevation raw file)
    COURSENAME_Lidar_Outer             (Outer Terrain file)
    COURSENAME_Outer_B                 (Outer BING Terrain satellite overlay)
    COURSENAME_Outer_G                 (Outer GOOGLE Terrain satellite overlay file)
    COURSENAME_Lidar_Surface_Outer     (Outer terrain elevation raw file)
    LightBlue                          (Special paint texture for lowering outer terrain)
    COURSENAME_Lidar_Scene             (The Scene file)
    COURSENAME_Layout                  (The course hole layout)
    README.txt                         (The read me text file)
